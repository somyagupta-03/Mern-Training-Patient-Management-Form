## Patient Management Form

A simple Patient Registry System built using HTML, CSS, Node.js, and Express.js. The application allows users to register patient details such as name, date of admission, and disease.
## Features
- Register new patients
- Store patient records in a text file
- View registered patients
- Simple and responsive UI

## Technologies Used
- HTML
- CSS
- JS
- Node.js
- Express.js
  

## How to Run

1. Install dependencies:

   npm install

2. Start the server:

   node app2.js

3. Open your browser and visit:

   http://localhost:3001
